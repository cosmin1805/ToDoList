- install laravel
- clone the repo/install it
- edit .env settings for yor database
- run insite a bash in the project directory: composer install
- then after making sure the database info is alright run: php artisan migrate
	- if you have a error try this https://stackoverflow.com/questions/42557693/laravel-pdoexception-could-not-find-driver
- if the migration works run: php artisan serve
